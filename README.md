# En Kanakku - என் கணக்கு 

Ena Kanakku is a personal expense analyser. The goal of the application is provide insights into your earning and spending patterns by analysing bank statement, credit card statements ...etc.,

This was created after using Firefly III and finding some features lacking and some overly complex.


## Development

This is a Python Django application. Use the docker-compose file for setting up the dev env.
